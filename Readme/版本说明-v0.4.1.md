# 版本说明-v0.4.1

对应对象：基于 Gemini 的电商 AI 出图工作台

## 版本信息

- 版本号：`v0.4.1`
- 安装器：`IMAGE-STUDIO-WINDOWS-0.4.1.exe`

## 版本概述

`v0.4.1` 是一个面向交付稳定性的补丁版本，重点修复 Windows 安装器在部分机器上启动后访问首页直接出现 `Internal Server Error` 的问题。

本说明面向对外交付与版本发布，重点描述这次补丁版的修复内容与交付影响，不重复展开 `v0.4.0` 已具备的完整功能清单。

## 本版本修复重点

### 1. 修复安装器首屏 500

- 修复发布包中遗漏 `sharp` 原生运行时 DLL 的问题
- 安装完成后首次访问首页不再因为图片处理依赖加载失败而直接报错
- 绿色发布目录与 Inno Setup 安装器使用同一套修复后的发布链路

### 2. 发布打包链路更稳

- 发布脚本会主动带上 `@img/sharp-win32-x64` 所需运行时文件
- 避免出现“本地开发正常、安装器运行报错”的依赖缺失场景
- 为后续继续发补丁版和安全发布包提供更稳定的基础

### 3. 继承 v0.4.0 功能基线

- 继续保留标准出图、提示词模式、参考图复刻、套图模式、亚马逊 A+ 图五种创作模式
- 继续保留历史记录、任务详情审核、通过项导出、模板中心、品牌库和飞书多维表格同步能力
- 继续支持版本化安装器命名、安全发布与局域网部署交付

## 适用对象

- 准备重新下载安装器的外部使用者
- 需要向协作者重新分发稳定安装包的项目维护者
- 已使用 `v0.4.0` 且遇到启动后首页 500 的团队

## 升级提醒

- 如果你之前安装的是旧版 `v0.4.0` 安装器，且启动后首页直接报错，建议直接改装为 `v0.4.1`
- 如果你当前使用的是源码运行或已手动修复过绿色发布目录，可按需要选择是否升级安装器
- 本次补丁主要修复发布产物稳定性，不改变已有业务配置结构；升级后仍需按原流程填写 Gemini 或 relay 配置，以及飞书接入参数
