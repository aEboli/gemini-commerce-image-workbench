# 当前端口问题的处理办法

你现在的截图说明：

- `3000` 端口已被进程 `13352` 占用
- `3001` 没有真正监听成功，只是有连接尝试

## 最快解决方法

直接双击：

`自动选择端口并启动.bat`

它会自动从 `3000~3005` 中找一个空闲端口启动，并自动打开浏览器。

---

## 如果你想手动清掉旧进程

先查看这个 PID 是什么：

```bat
tasklist | findstr 13352
```

如果确认是旧的 `node.exe` 或之前卡住的网站进程，就结束它：

```bat
taskkill /PID 13352 /F
```

然后再启动：

```bat
cd /d C:\Users\AEboli\Documents\Codex
npm run dev -- --hostname 127.0.0.1 --port 3000
```

---

## 如果你懒得判断进程

直接换端口：

```bat
cd /d C:\Users\AEboli\Documents\Codex
npm run dev -- --hostname 127.0.0.1 --port 3002
```

然后打开：

```text
http://127.0.0.1:3002
```
